# 環境

## 安裝指令(全域安裝)

請先檢查是否有安裝 `gulp` 及 `bower`

```
# gulp
$ npm install gulp -g

# bower
$ npm install bower -g
```

如果上述指令無法正常執行，請用以下並輸入系統密碼

```
# gulp
$ sudo npm install gulp -g

# bower
$ sudo npm install bower -g
```

---

## 使用指令

```
# 安裝相關套件
$ bower install
$ nmp install

# 開發環境輸入指令
$ gulp

# 發佈專案輸入指令
$ gulp build --env build
```
